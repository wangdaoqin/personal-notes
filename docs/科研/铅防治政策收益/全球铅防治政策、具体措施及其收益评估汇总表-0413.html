1998年-2000年 中国 全面禁用含铅汽油及国家标准(GB-I)实施 1. 2000年1月起全国禁产含铅汽油(含铅量需<0.013g/L)，7月全国禁售 ，北京(1997)与天津(1998)先行试点。[1, 2]；
2. 2000年起新产轿车须装配电喷及排气净化装置 [1]；
3. 提高有铅汽油消费税 [1]；
4. 快速提高MTBE和烷基化油产量以弥补辛烷值损失 [3]。		 1. 儿童血铅(BLLs)显著下降：禁铅1.5年后出现拐点 [4]，城市儿童血铅均值从2000年的121.49μg/L降至2013年的36.79μg/L [5]；
2. 成年人血铅下降：2000-2002年(GM 80.32μg/L)至2016-2018年(GM 21.57μg/L)下降约73% [6]；
3. 铅暴露相关的伤残调整寿命年(DALYs)显著下降，如3-5岁儿童从1666万降至41万 [8]，全国约减少1.81亿点IQ损失。[10]。
4. 环境空气铅排放减少约80-90% [7]。
5. 智力发育迟缓（MMR）发病率从13.87‰降至0.33‰。[4，6]
6. 天津气溶胶中 206Pb/207Pb 同位素比值显著上升,证明了机动车尾气贡献的下降。[12]		 1. 血铅几何均值(GM) [6, 8]；
2. 高血铅症(≥100μg/L)检出率：3-5岁儿童从2000年的41.3%降至2020年的5.3% [9]；
3. 智商(IQ)损失评估：导致中国城市儿童损失约3600万智商点，农村1.44亿点 [10]；
4. 伤残调整寿命年(DALYs)：儿童铅暴露DALYs约为0.10a [11]。		 [1-16]
2006年-2021年 中国 铅蓄电池管理与重点重金属排放总量控制 1. 2016年发布《废电池污染防治技术政策》，实施废铅蓄电池全过程监管，禁止人工露天拆解，严禁私自破损。
2. 废铅蓄电池利用企业的废水、废气排放应执行《再生铜、铝、铅、锌工业污染物排放标准》（GB 31574）。 [17, 18]； 执行《铅、锌工业污染物排放标准》（GB 25466-2010）[59]
3. 2011年《重金属污染综合防治“十二五”规划》实施总量控制 [19]；
4. 2019年实施《重点重金属污染物排放总量控制管理办法》，比2013年减排10% [20]。
5. 实施《重点重金属污染物排放总量控制管理办法》，要求2020年重点行业排放量比2013年下降10% [20]。		 1. 截至2015年底，5种重点重金属排放总量比2007年下降27.7% [21]；
2. 行业绿色化：累计淘汰铅冶炼产能381万吨、铅蓄电池9622万千伏安时 [21]；
3. 受污染耕地安全利用率达到87.26% [22]；
4. 突发环境事件从每年十余起降至不足3起 [21]。		 1. 重点重金属排放量核算比(含废气、废水中铅总量) [19, 20]；
2. 工业大气铅排放量及农用地土壤铅累积水平 [9]；
3. 集中式饮用水水源地达标率 [19]。		 [9, 13, 17-26]
2006年-2024年 中国 临床干预、涂料限制及土壤/地下水综合防治 1. 2006年《儿童高铅血症和铅中毒预防指南》规定诊断标准(≥200μg/L)及螯合治疗规范 [27]；
2. 1986年规定玩具漆膜可溶性铅限值250 ppm。2003年更新涂料标准，限制可溶性铅浓度<90ppm [7]；制定肥料、食品及儿童用品铅限值。[5]
3. 严格限制城市及郊区新建铅冶炼厂(国家发改委，2007)。[2]
4. 实施“十条土壤规则”（2016）[58]和2019年《土壤污染防治法》施行，划定耕地环境类别 [28]；
5. 2022年《关于进一步加强重金属污染防控的意见》，重点行业排放比2020年下降5% [25]。
6. “十四五”规划加强耕地源头控制，实施重金属排放总量减量替代（1.2:1）。[25]		 1. 2016-2021年全国大部分地区儿童血铅低于3.5μg/dL [4]；
2. 受污染耕地和污染地块安全利用率实现“双90%” [28]；
3. 规范诊疗行为，降低了盲目螯合治疗的副作用风险 [29]；
4. 湖南重点区域水环境达标率从2016年的75%升至2020年的83.3% [30]。		 1. 危害商数(HQ)与IEUBK模型预测 [4]；
2. 静脉血铅水平及驱铅试验指数(I值) [27, 31]；
3. 涂料可溶性铅浓度 [7]；
4. 重点区域监测断面重金属因子质量达标率 [30]。		 [4, 7, 14-16, 25, 27-34]
1970年代-2021年 美国 多领域铅管控(汽油、涂料、饮用水) 1. 1973年EPA颁布削减标准，1975年强制淘汰含铅汽油 [35, 36]；
2. 1978年禁止住宅涂料总铅超过600ppm，后降至90ppm [37, 38]；
3. 2021年更新儿童血铅参考值(BLRV)至3.5μg/dL [8]；
4. 1982年实施“铅信用额度交易”降低炼油成本 [39]。
5. 通过NHANES进行大规模生物监测，多次下调血铅参考值（40 μg/dL降至3.5 μg/dL）。		 1. 儿童血铅水平下降70%以上 [36]：1-5岁儿童血铅95分位值从1999-2000年的7.00μg/dL降至2015-16年的2.76μg/dL [37]；儿童（<6岁）BLL从1970s的141-158 μg/L显著降至2013-2014年的8.4 μg/L。
2. 健康获益：成人血铅降至10μg/L以下每年减少约41.2万人死亡 [6]；
3. 犯罪率下降. ：每年减少因暴力犯罪和IQ损失导致的经济支出，若BLL降至10 μg/L以下，每年可减少约41.2万名成人死亡，滞后期19-23年 [40]。		 1. 智商(IQ)损失：血铅每升高10μg/dL，IQ下降约2.5点 [39, 41]；
2. 伤残调整寿命年(DALYs)：2017年铅暴露导致全球2440万年健康寿命损失 [37]；
3. 统计生命价值(VSL)：估值每人300万美元 [39]。		 [5, 6, 8, 9, 35-44]
1970年代-2024年 欧盟/欧洲国家 (含瑞典、英国) 含铅汽油、涂料、渔具及湿地铅弹禁令 1. 瑞典分步降限值(1.2至0.15g/L)并实施差别税 [39]；
2. 1987年英国禁止0.06g至28.35g的钓鱼铅坠 [45, 46]；
3. 2023年禁止在湿地使用铅弹(REACH法规) [46]；
4. 2025年起航空燃料四乙基铅须经授权使用 [46]。		 1. 瑞典儿童血铅几何均值从1978年的67μg/L降至2007年的13μg/L [47]；
2. 物种恢复：英国疣鼻天鹅死亡率从0.34降至0.06，种群数量增加100% [45, 46]；
3. 减排预期：未来20年预计减少63万吨铅排放 [46]。
4. 空气铅浓度下降85%（0.31降至0.045 μg/m³）。		 1. 疣鼻天鹅血铅浓度及死亡原因分类 [45]；
2. 人体生物监测(HBM)指标 [46]；
3. 饮食铅摄入量：日均从68.7μg/d降至35.7μg/d [48]。		 [39, 43, 45-49]
1970年-1986年 日本 早期汽油脱铅策略 1. 1971年修订《有毒有害物质控制法执行令》，限值从1.3cc/L降至0.3cc/L [50]；
2. 通过不同颜色标签及统一加油枪喷嘴防止误加油 [39]；
3. 1986年后不再生产或使用有铅汽油 [39, 50]。		 1. 有效应对了东京牛込柳町严重的血铅超标事件 [50]；
2. 促进了日本汽车工业的全球竞争力 [50]；
3. 1980年代初仅剩1-2%汽油含铅 [39]。		 1. 大气中铅及一氧化碳(CO)水平 [39, 50]；
2. 居民血铅水平监测 [50]。		 [39, 50]
1980年代-1995年 斯洛伐克 旧车占比高环境下的脱铅技术 1. 炼油厂升级建设异构化单元 [39]；
2. 研发ANABEX-99代铅添加剂，为软阀座旧车提供保护 [39]；
3. 对无铅汽油实施优惠定价 [39]。		 1. 无铅汽油市场份额从1992年的6%升至1995年的100% [39]；
2. 实现全境脱铅单位成本约0.02美元/升 [39]。		 1. 汽油含铅量监测(0.7g/L降至0) [39]；
2. 无铅汽油普及率 [39]。		 [39]
1990年-1995年 泰国 快速淘汰与炼油厂重组 1. 世界银行贷款支持Bangchak炼油厂脱铅工艺改造 [39]；
2. 1995年底全面禁铅 [39]；
3. 建立环境空气监测网络及机动车排放检测标准 [39]。		 1. 四年内完全消除车辆铅排放 [39]；
2. 显著降低了曼谷等城市的人群铅暴露水平(此前曼谷儿童血铅曾达40μg/dL) [39]。		 1. 炼油厂生产效率及投资回报率 [39]；
2. 儿童血铅水平(BLLs) [39]。		 [39]
1989年-2012年 美国 (新罕布什尔州) 限制含铅渔具保护野生动物 1. 建议更换重量<28.4g的含铅渔坠和夹具头 [51]；
2. 建立人工嵌套平台和公众教育 [51]。		 1. 物种保护：若移除铅渔具压力，普通潜鸟种群可增加43% [51]；
2. 含铅渔具曾占成年潜鸟死因的48.6%，政策有效降低了致死率 [51]。		 1. 种群增长率(λ)：铅渔具致死导致增长率下降1.4% [51]；
2. 栖息地承载力 [51]。		 [51]
1976年-2010年 加拿大 涂料铅含量限制演变 1. 1976年限值5000mg/kg，2005年强制限制为600mg/kg，2010年加严至90mg/kg [38]。 1. 1993-1995年儿童血铅均值降至15.7μg/L [52]；
2. 1999年评估政策收益占GDP的3.2% [41]。		 1. 涂料总铅含量(mg/kg) [38]；
2. 政策收益/GDP比率 [41]。		 [38, 41, 52]
2002年-2021年 国际组织 (UNEP, WHO, CDC) 全球消除含铅涂料行动及健康指南更新 1. 帮助80多个国家淘汰含铅燃料 [53]；
2. 2021年最后一个国家阿尔及利亚完成淘汰 [50, 53]。
3. 成立全球消除含铅油漆联盟（LPA），推动各国将油漆总铅限制在90 ppm以下。
4. 更新儿童铅中毒防治指南，将“关注水平”下调（2012年降至50 μg/L，2021年CDC降至35 μg/L）。		 1. 每年产生2.45万亿美元经济收益(约全球GDP的4%) [41, 53]；
2. 每年减少120万例过早死亡，其中12.5万为儿童 [53]；预计减少全球每年约60万例新增MMR。
3. 每年产生约9770亿美元的IQ提升生产力效益 [54]。
4. 减少全球每年约106万例归因于铅暴露的死亡。		 1. 剂量-反应曲线显示低浓度（<10 μg/dL）下下降斜率更陡。2017年全球铅暴露导致2440万DALYs损失。 [53]；
2. 犯罪避免数量(5800万起) [53]；
3. 成本收益比(至少10:1) [41]。		 [5, 35, 41, 50, 53-57]
