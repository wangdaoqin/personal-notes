R语言基础教程

简介

R语言是一种用于统计计算和图形绘制的编程语言和软件环境，广泛应用于数据分析和统计建模领域。

基本语法

1. 变量赋值

# 使用赋值运算符 <-
x <- 10
y <- "Hello, R!"

# 也可以使用 = 号
z = 25

2. 数据类型

# 数值型
num <- 3.14

# 字符型
char <- "R语言"

# 逻辑型
flag <- TRUE

# 查看数据类型
class(num)

3. 向量和数组

# 创建向量
vec <- c(1, 2, 3, 4, 5)

# 序列
seq_vec <- 1:10

# 重复
rep_vec <- rep(c(1, 2), times = 3)
提示：R语言中的索引从1开始，而不是0！

4. 数据框

# 创建数据框
df <- data.frame(
    name = c("Alice", "Bob", "Charlie"),
    age = c(25, 30, 35),
    score = c(85, 92, 78)
)

# 查看数据框结构
str(df)

# 查看前几行
head(df)

数据处理基础

安装和加载包

# 安装包
install.packages("dplyr")

# 加载包
library(dplyr)

数据筛选和变换

# 使用dplyr进行数据操作
result <- df %>%
    filter(age > 25) %>%
    mutate(score_level = ifelse(score >= 90, "高", "中"))

数据可视化

# 绘制直方图
hist(df$score, main = "分数分布", xlab = "分数", col = "skyblue")

# 绘制散点图
plot(df$age, df$score, main = "年龄与分数关系", xlab = "年龄", ylab = "分数")

# 使用ggplot2绘图
library(ggplot2)
ggplot(df, aes(x = age, y = score, color = name)) +
    geom_point(size = 3) +
    labs(title = "散点图示例")
注意：R语言区分大小写，Sum()和sum()是不同的函数！

常用函数

  • mean() - 计算平均值
  • sd() - 计算标准差
  • summary() - 生成描述性统计
  • cor() - 计算相关系数
  • lm() - 线性回归分析

总结

R语言是一门功能强大的统计分析语言，适合数据科学家、统计学家和研究员使用。通过学习和掌握R语言，你可以高效地进行数据处理、统计分析和可视化工作。
